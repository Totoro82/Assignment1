### How to run:
1. open a terminal on project root
2. to compile type `make`
```sh
make 
```
3. to run type `make run < N >` (with N being the random input number)

example with N = 100
```sh
make run 100
```
4. in case you want to remove the executable file, simply run:
```sh
make clean
```
